/movies

Return the list of all movies

Request Body data

None

Response Body data

JSON object holding data about all the movies: Array of:
{_id: Number, Title: String, Description: String, Genre: [ref_id: 'Genres'], Director: ref_id: 'Directors', Actors: [String], ImagePath: String, Featured: Boolean }

Response Body data

{_id: Number, Title: String, Description: String, Genre: [ref_id: 'Genres'], Director: ref_id: 'Directors', Actors: [String], ImagePath: String, Featured: Boolean }
